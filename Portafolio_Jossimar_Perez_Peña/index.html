Encriptador de texto

Realizado en VScode mediante estructura HTML y estilo CSS, proyecto realizado durante curo de Java scrip en Alura Latam. Este proyecto consiste en una aplicación web que permite encriptar y desencriptar mensajes de texto utilizando un algoritmo simple de sustitución de caracteres. La interfaz está diseñada para ser intuitiva y funcional, con un área de texto de entrada y salida alineadas en columnas, botones para ejecutar la encriptación/desencriptación, y un botón de copiar resultados al portapapeles. El diseño responde a las necesidades de claridad, facilidad de uso, y cumple con ciertas reglas, como aceptar solo letras minúsculas sin acentos.
